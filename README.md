# LogSwap — セットログ交換アプリ（デモ）

「セットログ（その日の断片的な日常ログ）」を交換する相手を見つけるための、Tinder 風スワイプ UI のデモアプリです。

- 画面に他のユーザーの **セットログ** が1枚ずつ表示されます
- **右スワイプ / ◯** … ログを交換したい
- **左スワイプ / ✕** … 今回は見送る
- おたがいが ◯ を出すと **「ログ交換 成立」**。相手のセットログ（時系列のミニ Vlog）が見られます

> [!NOTE]
> このアプリは **セットログを交換するためのツール** です。
> マッチングアプリ・出会い目的のサービスではありません。
> 性別・年齢でのフィルタリングなど、異性紹介に該当しうる機能は意図的に持たせていません。

## デモを触る

GitHub Pages で公開しています（リポジトリの Settings → Pages から有効化）。

```
https://<ユーザー名>.github.io/<リポジトリ名>/
```

ローカルで動かす場合（ビルド不要・静的ファイルのみ）:

```bash
# どれか1つ
python -m http.server 8000
# または
npx serve .
```

ブラウザで `http://localhost:8000` を開きます。
スマホ表示が前提のレイアウトですが、PC では矢印キー（←／→）でもスワイプできます。

## 使い方

| 操作 | 内容 |
| --- | --- |
| カードを右へドラッグ / ◯ ボタン / → キー | ログを交換したい |
| カードを左へドラッグ / ✕ ボタン / ← キー | 見送る |
| ⌄ ボタン | そのユーザーのセットログを詳しく見る |
| 🤝 ボタン | 交換が成立したログの一覧 |

相互に ◯ になると交換成立のオーバーレイが出て、相手の1日のログ（時刻ごとの 2 秒クリップ風の絵文字 + ひとこと）を閲覧できます。

## 技術構成

- 素の **HTML / CSS / JavaScript** のみ。ビルド・依存パッケージなし
- バックエンドなし。ユーザーは `js/data.js` のモックデータ
- マッチングは擬似的（`likesBack: true` のユーザーに ◯ を出すと成立）
- スワイプは Pointer Events で実装

```
.
├── index.html      # 画面構成
├── css/style.css   # スタイル
└── js/
    ├── data.js     # モックユーザー & セットログ
    └── app.js      # スワイプ / マッチ / 表示ロジック
```

## このデモから本番にするには

- `js/data.js` のモックを API 取得に置き換え
- 相互 ◯ の判定をサーバー側のマッチングに置き換え（双方が ◯ で成立）
- セットログ本体（動画など）の保存・配信基盤を追加
